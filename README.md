# Getting Started

JetBrains task

## How to build

> gradle bootWar

## How to run

Go to build -> libs folder

> java -jar oauth-1.0.0.war


## Available endpoints

##### Home page
>/home
##### Sign in page
>/signin
##### Endpoint to shut down the server
>/stop
